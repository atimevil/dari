# 다리 (DARI) — 유학생 비자·체류 AI 도우미

> 내 체류상황을 아는 AI가, 내 모국어로, 공식 근거와 함께 24시간 안내합니다.
> 1박 2일 해커톤 MVP. 설계는 [`SPEC.md`](./SPEC.md), 발표 자료는 [`BRIEFING.md`](./BRIEFING.md) 참고.

## 무엇인가
중국·베트남 등 외국인 유학생이 비자 연장·외국인등록·시간제취업허가 같은 **체류 행정**을,
**개인 체류상황 기반 + 다국어 + 공식 출처 인용**으로 안내하는 챗봇.
시중 앱은 한국어·정보제공 중심 → 다리는 **개인화·다국어·출처기반·비자 특화**가 차별점.

## 핵심 기능 (MVP)
- 🗣️ 개인화 다국어 Q&A (한국어·中文·Tiếng Việt·English)
- 🔗 공식 출처(하이코리아 등) 기반 답변 + 출처 인용
- ✅ 절차 체크리스트 / 📄 필요 서류 / ⏰ 비자 만료 D-day
- ⚠️ 고위험·미지원 사안은 출입국·외국인청 연결(에스컬레이션) — 환각 차단

## 실행 방법
```bash
# 1) 의존성 설치
npm install

# 2) API 키 설정 — .env.local 파일 생성 후 키 입력
#    (.env.example 참고)
#    OPENAI_API_KEY=sk-...

# 3) 개발 서버 실행
npm run dev
# → http://localhost:3000
```

> OpenAI API 키는 https://platform.openai.com/api-keys 에서 발급.
> 모델은 기본 `gpt-4o`. 비용 우선이면 `.env.local`에 `DARI_MODEL=gpt-4o-mini`.

## 데모 시나리오
1. 첫 화면에서 **🎬 데모 프로필 채우기** 클릭 (베트남 유학생 Linh, D-2, 만료 D-43, 알바 중)
2. **시작하기** → 언어가 Tiếng Việt로 설정됨
3. 질문: *"겨울방학에 아르바이트 시간을 늘릴 수 있나요?"*
4. 베트남어 답변 + 절차 + 출처 + 비자 만료 D-day 확인
5. 고위험 질문 *"비자를 D-10으로 바꾸려면?"* → ⚠️ 에스컬레이션 시연
6. 상단 언어 토글로 中文/한국어 즉시 전환

## 구조
```
src/
  app/
    page.tsx              메인 UI (프로필 폼 + 채팅 + 언어 토글 + D-day)
    api/chat/route.ts     OpenAI(GPT) 호출 (시스템 프롬프트 + JSON 파싱)
    layout.tsx, globals.css
  lib/
    knowledge.ts          큐레이션 공식 참고자료 (벡터DB 없이 프롬프트 주입)
    prompt.ts             시스템 프롬프트 빌더 (정확성·에스컬레이션 강제)
    types.ts              공통 타입
public/logo-icon.svg      브랜드 로고 (assets/ 에 가로형·대안 로고도 있음)
```

## ⚠️ 중요 — 데이터 검증
`src/lib/knowledge.ts`의 참고자료는 **프로토타입용 예시**입니다.
실제 서비스 전 반드시 **하이코리아·법무부 출입국 공식 원문 최신본**으로 교체/검증하세요.
출입국 정보는 오안내 시 학생에게 치명적이므로, 행정사 검수 + 공식 출처 grounding을 전제로 합니다.
